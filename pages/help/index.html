Background

We have been working on this project of Web Applications since our 11th(2020-21) grade, since the time our dear teacher Mrs. Pooja Garg intimated us about it. Super-excited, we started working on it right away. We decided the topic and the journey began. By the end of the year(11th) we had completed it with the knowledge we had at that point of time, and then later we did not touch this project until we started our 12th grade.

We almost forgot about this project, until we were reminded about it for project submission. Later, when we reflected on this project, we found lacunas as it was based on the knowledge that we had earlier. By now we had attained quite a bit of experience on web design fundamentals, and wanted to do a bit extra.

We decided to rework on the same project, change the design and make it aesthetic. First we thought to use bootstrap for everything but then we realized that using bootstrap is not a good idea as we were skipping a lot of parts of actual coding in the design and were relying on framework that bootstrap offers. So, we decided to code it ourselves (for the most part), and make the best out of our skill sets, and knowledge we had gathered to create "GAME ARENA" !

A gaming arena where you can enjoy the game the way you did in your childhood-with sheer pleasure! Hope you found the games exiting and had fun. A special note of thanks to Pooja ma'am for teaching us to create a project on such a level. On this roller-coaster of a ride we learned a lot and hope to continue learning. Hope you enjoy and continue enjoying. Let the journey begin!
